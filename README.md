# Ashbi Design (Digital Agency & Development Hub)

**The central portfolio and main web presence for Ashbi Design, a digital agency specializing in high-performance web applications, bespoke software, and AI integration.**

Ashbi Design is the digital storefront demonstrating our ability to bridge beautiful front-end aesthetics with deep, agentic automation. This repository serves as the professional face of our consulting and development services, available at [Hub.ashbi.ca](http://Hub.ashbi.ca).

## 🚀 Vision: AI-First Consulting

While our SaaS products (like the Pi Website Builder and GlowOS Engine) serve mainstream consumers, **Ashbi Design** remains the premium touchpoint for enterprise clients and bespoke integrations. 
- **The Pitch:** We build the very tools that build websites. 
- **The Execution:** High-converting, emotionally resonant landing pages (like our `arcan-painting-emotional` template) paired with complex backend orchestrations (like `fleetflow-mvp` and `SealSend`).

## 🛠 Tech Stack

- **Framework:** Next.js (App Router)
- **Styling:** Tailwind CSS + custom high-end animations (Framer Motion)
- **CMS / Data:** Sanity.io or integrated Prisma/Supabase for dynamic portfolio management
- **Deployment:** Vercel for Edge delivery

## ⚡ Getting Started

```bash
# Clone the repository
git clone https://github.com/camster91/Ashbi-Design.git
cd Ashbi-Design

# Install dependencies
npm install

# Setup environment variables
cp .env.example .env.local

# Run development server
npm run dev
```

Visit `http://localhost:3000`.
